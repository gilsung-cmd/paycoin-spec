페이코인, 1분기 결제 거래량 전년 동기 대비 230% 성장
연합뉴스 2026.04.30
원문 보기 ↗

페이코인(대표 한정수)이 2026년 1분기 결제 거래량이 전년 동기 대비 230% 성장했다고 30일 밝혔다. 이는 온라인 결제 가맹점 확대와 오프라인 결제 활성화에 따른 결과다.

페이코인 관계자는 "올해 들어 주요 편의점 체인과 외식 프랜차이즈를 중심으로 가맹점이 빠르게 확대되고 있으며, PCI 토큰 결제의 편의성을 경험한 사용자들의 재이용률이 높게 나타나고 있다"고 말했다.

회사 측은 2026년 하반기까지 가맹점 3만 개소 확대를 목표로 마케팅 투자를 지속할 계획이라고 밝혔다. PCI 토큰 시세도 이번 발표에 힘입어 소폭 상승세를 보였다.

⚙️ Apps Script 코드
아래 코드를 복사해서 Google Apps Script에 붙여넣으세요.
이 코드는 현재 뷰어의 화면 구성을 자동으로 반영합니다.
📋 복사할 코드
로딩 중...
📌 적용 방법
1
Google Sheets → 확장 프로그램 → Apps Script 열기
2
기존 코드 전체 삭제 후 위 코드 붙여넣기
3
배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자 → 배포
4
배포 URL을 복사해서 config.json 탭에 입력
🔧 config.json 설정
HTML 파일과 같은 폴더에 config.json 파일을 만들고 아래 내용을 넣으세요.
📋 config.json 템플릿
{
  "apiUrl":      "← Apps Script 배포 URL 붙여넣기",
  "projectName": "← 프로젝트/앱 이름",
  "primaryColor": "← 브랜드 컬러 HEX (예: #1A56DB)"
}
💡 각 항목 설명
apiUrl — Apps Script 배포 후 받은 URL
projectName — 뷰어 제목과 로고에 표시될 이름
primaryColor — 버튼, 강조색 등에 사용할 브랜드 컬러
⚠️ 로컬(파일로 직접 열기)에서는 config.json이 로드되지 않습니다.
이 경우 QnA 사이드바 하단 ⚙️ 버튼에서 URL을 직접 입력하세요.
